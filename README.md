# Prerequirement
## Install Conda

Conda install guide can be found in https://www.anaconda.com/docs/getting-started/getting-started

# Step

## Cloning repo

```
conda env create -f environment.yml
```

## Building envonment

```
conda env create -f environment.yml
```

## Activate envonment

```
conda activate ./.env
```

## Run the program

```
python ./main.py
```
